# Bank Management System

## About Project

Bank Management System is a simple console-based Java project developed using Core Java and Object-Oriented Programming concepts.  
This project simulates basic banking operations like account creation, login authentication, deposit, withdrawal, balance inquiry, and transaction history.

The main purpose of this project is to practice:
- OOP concepts
- ArrayList
- Exception Handling
- User Input Handling
- Console-based Menu System
- Real-world project structure

This project was developed as an internship task.

---

# Features

- Create New Bank Account
- Secure Login using Account Number and PIN
- Deposit Money
- Withdraw Money
- Check Current Balance
- View Transaction History
- Logout System
- Input Validation
- Exception Handling using try-catch
- Console Menu Interface
